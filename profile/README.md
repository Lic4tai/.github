# Licitaí

**Pare de caçar editais. Comece a receber oportunidades prontas.**

Software para **monitoramento inteligente de licitações públicas** no Brasil, conectado ao **PNCP** — feito para quem quer **ganhar tempo**, **não perder oportunidades** e **vender mais para o governo**.

---

### 🚀 O problema

Todos os dias, novas licitações são publicadas.  
E a maioria das empresas:

❌ Perde oportunidades por falta de tempo  
❌ Depende de buscas manuais e cansativas  
❌ Descobre editais tarde demais  

---

### 💡 A solução

Com o **Licitaí**, você não procura mais.  
Você **define o que vende** — e nós fazemos o resto.

```text
// Você cria seus filtros (monitores inteligentes)
// A gente acompanha o PNCP em tempo real
// Encontrou algo relevante?
→ Você recebe o alerta direto (e-mail, em breve mais canais)
```
